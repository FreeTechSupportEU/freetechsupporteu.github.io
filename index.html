How to Apply

We select projects based on social impact. A limited number of new partnerships are selected for this year, where technical skills and cloud resources can create the greatest community change.

To apply, email askfreetech@gmail.com by providing the following details:

  • Mission: Who you help and how.
  • Tech Challenge: What is preventing you from growing.
  • Social Impact: How improvements will change beneficiaries' lives (e.g., "manage 30% more food").
  • Project Scale: Number of volunteers/staff who will use the system.

Note: We only accept applications from officially registered nonprofit organizations
